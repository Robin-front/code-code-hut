

### Math

- ceil

```js
// _ceil(6.004) => 7
// _.ceil(6.004, 2) => 6.01
// _ceil(6040, -2) => 6100
```

- floor

```js
_.floor(4.006);
 // => 4

_.floor(0.046, 2);
 // => 0.04

_.floor(4060, -2);
 // => 4000
```

- round

```js
_.round(4.006);
 // => 4

_.round(4.006, 2);
 // => 4.01

_.round(4060, -2);
 // => 4100
```
